# 《构建可见世界：APSAL 元素摄影法》

**副标题：从视觉元素、关系语法到可复现的 AI 摄影系统**  
**英文名：Building Visible Worlds: The APSAL Elemental Method for AI Photography**
**作者：“APSAL 创始人” HenyJone / HenyJone, Founder of APSAL**

本书是 APSAL 的开放方法论专著。它讨论的不是如何写出更长的提示词，而是如何把摄影创作理解为一种世界构建活动：识别视觉元素，声明元素之间的关系与约束，建立可持续的世界状态，再通过镜头把这个世界转化为一组相互独立、彼此连续的图像。

> AI 摄影的本质，不是描述一张图，而是用结构化、元素化的视觉语言，构建让图像得以成立的世界。
