# APSAL 元素摄影法术语表

| 术语 | 定义 |
|---|---|
| 元素 Element | 可独立命名、改变、锁定、复用或检验的视觉单位。 |
| 关系 Relation | 元素之间的空间、因果、依赖或意义连接。 |
| 约束 Constraint | 声明必须保持、不得发生或仅在特定条件下变化的边界。 |
| 世界 World | 元素、关系、约束、状态和可观看视点构成的统一现实。 |
| 世界状态 State | 某一时刻人物、空间、事件、物件、光线和情绪的组合。 |
| DNA | 具有稳定身份、版本、依赖、权利和 QA 元数据的可复用视觉知识。 |
| Job | 对共享世界的一次独立观看与生成任务；默认对应一张完成图。 |
| 身份锁 Identity lock | 优先于提示词修辞、用于保护主体稳定身份的不可变约束。 |
| 单变量变体 | 只改变一个已声明字段、其余非目标字段保持锁定的变体。 |
| 连续性账本 | 记录镜头之间世界状态变化的结构化记录。 |
| Canonical core | 不含平台专用参数的规范化核心资产。 |
| Adapter | 将规范资产映射到具体模型或平台的适配层。 |
| 静态验证 | 对 Schema、依赖、版本、权利、摘要和输出规则的自动检查。 |
| 视觉 QA | 基于实际生成证据，对身份、解剖、光、空间、连续性和审美的人工判断。 |
| 可复现性 | 创作意图、结构、过程和判断标准可以被重新执行，而非像素完全相同。 |
