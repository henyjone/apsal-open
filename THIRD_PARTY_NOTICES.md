## MOSA

APSAL Studio 0.3 and Engine 0.16 incorporate and adapt local media-library
designs from MOSA at the pinned source commit below:

- Project: `fengseekling-coder/mosa`
- Source commit: `38be7f4c951b75ea032392a27abea1d2170b6bac`
- Source URL: <https://github.com/fengseekling-coder/mosa/commit/38be7f4c951b75ea032392a27abea1d2170b6bac>
- License: MIT

The APSAL adaptation uses a rebuildable SQLite project/asset index,
content-addressed generated-output storage, project cards, search, and
thumbnail-oriented browsing. It is implemented inside the APSAL Python Engine
and Electron Studio. MOSA's standalone web server, Node 22 runtime, Cowart
integration, Codex history watcher, and broad generated-image importer are not
included.
